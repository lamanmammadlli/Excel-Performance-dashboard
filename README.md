

Excel Interactive Performance Dashboard

Overview

An interactive Excel dashboard analyzing sales, profit, and product performance for a cookie company. Built using PivotTables, PivotCharts, and slicers to explore trends by date, country, and cookie type.

Features

Interactive filters (date, country, product)

Monthly profit and units sold trends

Profit by market and cookie type

Automatic updates based on slicers
